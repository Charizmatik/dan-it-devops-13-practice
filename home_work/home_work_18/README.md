# Homework 18

## AWS вправа

Файл з відповіддю: [hw_18.md](hw_18.md)

## Завдання 1: ECR

Створіть власний репозиторій ECR.

Зберіть власний образ на своєму комп'ютері.

Авторизуйтесь у свій репозиторій ECR.

Push створений образ у репозиторій ECR.

## Завдання 2: RDB

Створіть базу даних MySQL у сервісі AWS.

Підключіться до бази даних з комп'ютера за допомогою утиліти `mysql`.

Видаліть базу даних.

## Завдання 3: Lambda

Напишіть функцію AWS Lambda, яка буде запускатися за розкладом (`cron`) о 12:00 дня і зупинятиме всі EC2-екземпляри, що працюють із певним тегом.

Використайте мову Python та бібліотеку `boto3` для реалізації функції.

Після перевірки роботи видаліть Lambda-функцію та пов'язану IAM-роль, створену для неї.

## Формат здачі ДЗ

### Завдання 1

1. Скрін зі створеним ECR.
2. Код з Dockerfile, в якому буде, наприклад, встановлення вебсервера Apache, додавання одного рядка тексту в `index.html`, запуск `httpd`.
3. Скрін з описом, як використовувати ECR (`Push commands`).
4. Скрін з логіном з командного рядка в ECR.
5. Скрін з білдом імеджа з Dockerfile.
6. Скрін з пушем імеджа в створений ECR.
7. Скрін з видаленням ECR.

### Завдання 2

1. Скрін зі створеним AWS MySQL database.
2. Скрін з підключенням з власного ПК до створеного AWS MySQL.
3. Скрін з видаленням AWS MySQL database.

### Завдання 3

1. Скрін з EventBridge з відповідними тригерами.
2. Скрін зі створеною IAM-роллю.
3. Скрін зі змістом IAM-ролі.
4. Скрін зі створеною/створеними EC2 з відповідним тегом, за яким фільтр Lambda відпрацьовує.
5. Код Lambda Function на Python з використанням бібліотеки `boto3`.
6. Скрін, як відпрацьовує Lambda. Чекати 12:00 не обов'язково, можна поставити інший час.
7. Скрін з видаленням Lambda Function й IAM-ролі.
