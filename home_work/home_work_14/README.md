# Homework 14

## Частина 1

Створіть файл `docker-compose.yml` на основі Homework 13.

У цьому файлі потрібно налаштувати сервіс `app`, використовуючи Docker-образ із Homework 13. Образ для сервісу `app` має створюватися динамічно за допомогою `docker compose`.

## Частина 2

Додатково налаштуйте сервіс `nginx` і сконфігуруйте його як зворотний проксі (`reverse proxy`) до сервісу `app`, щоб він обробляв запити через HTTP-порт `8080`.

Конфігураційний файл `nginx` має знаходитись за шляхом `/etc/nginx/conf.d/default.conf` всередині контейнера.

У підсумку потрібно завантажити (`push`) до GitHub:

- `docker-compose.yml`
- конфігураційний файл `nginx`
- вихідний код вашого застосунку
- скріншоти, що демонструють працездатність налаштування

## Формат здачі ДЗ

1. Код `docker-compose.yml`
2. Код `Dockerfile`
3. Код `nginx.conf`
4. Скріншоти запуску й роботи сервісів
5. Скріншоти результату роботи сервісів у браузері
