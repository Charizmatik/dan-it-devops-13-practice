# Завдання: ABC
## Опис структури класів

### Клас Alphabet (Алфавіт) характеризується:
- Мовою
- Списком літер

### Для об’єкта класу Alphabet можна:
- Вивести всі літери алфавіту
- Порахувати кількість літер у алфавіті

### English Alphabet (Англійський алфавіт) має такі властивості:
- Мова
- Список літер
- Кількість літер

### Для англійського алфавіту (EngAlphabet) можна:
- Порахувати кількість літер
- Визначити, чи належить літера англійському алфавіту
- Отримати приклад тексту англійською мовою

### Завдання
### Клас Алфавіт
1. Створіть клас Алфавіт.
2. Створіть метод __init__(), у якому будуть визначені два атрибути:
- lang — мова
- letters — список літер

### Початкові значення цих властивостей передаються через параметри методу.
- Створіть метод print(), який виводить літери алфавіту на екран.
- Створіть метод letters_num(), який повертає кількість літер у алфавіті.

### Клас Англійський алфавіт (EngAlphabet)
1. Створіть клас Англійський алфавіт (EngAlphabet), який успадковує клас Алфавіт.
2. Створіть метод __init__(), у якому викликається батьківський метод __init__(). До нього передаються позначення мови (наприклад, En) та рядок, що містить усі літери алфавіту будуть передані йому, як параметри.
3. Створіть приватний статичний атрибут _letters_num, який зберігатиме кількість літер в алфавіті.
4. Створіть метод s_en_letter(), який приймає літеру як параметр і визначає, чи належить ця літера англійському алфавіту.
5. Перевизначте метод letters_num() — тепер він має повертати значення атрибуту _letters_num у поточному класі.
6. Створіть статичний метод example(), який повертатиме приклад тексту англійською мовою.

## Тести (main)
1. Створіть об’єкт класу EngAlphabet.
2. Виведіть літери алфавіту для цього об’єкта.
3. Виведіть кількість літер у алфавіті.
4. Перевірте, чи належить літера 'F' англійському алфавіту.
5. Перевірте, чи належить літера 'Щ' англійському алфавіту.
6. Виведіть приклад тексту англійською мовою.
