Завдання
Створіть VPC.

Створіть дві підмережі (subnet groups): публічну та приватну 

Створіть один екземпляр EC2 у публічній підмережі та один EC2 у приватній підмережі.

Переконайтесь, що EC2-екземпляр у публічній підмережі доступний з інтернету та до нього можна під’єднатися через SSH.

Переконайтесь, що EC2-екземпляр у приватній підмережі не доступний з інтернету, але доступний із EC2-екземпляра публічної підмережі.

Переконайтесь, що EC2-екземпляр у приватній підмережі має доступ до інтернету.

Формат здачі ДЗ:

1. Скрін з створеним VPC

2. Скріни з створеними сабнетами

3. Скріни з створеними ЕС2

4. Скріни з створенням всього необхідного для доступу до Інтернету

5. Скрін з підключенням по SSH до EC2, яка в публічній підмережі

6. Скрін з підключенням по SSH з EC2, яка в публічній підмережі до ЕС2, яка в приватній підмережі

7. Скрін з пінгом google.com з EC2, яка в приватній підмережі
